* Recognizes garments and accessories in images or videos
* Provides personalized styling recommendations based on user preferences and trends
* Integrates with e-commerce platforms for seamless purchasing experiences

## Installation 💻

To get started, follow these steps:

1. Clone this repository: `git clone https://github.com/your-username/stylist-brain.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Set environment variables: `cp .env.example .env` (update as needed)
4. Run the application: `sh run.sh`
